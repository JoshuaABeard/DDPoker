Online Play - DD Poker Community Edition
Play Online With Friends and Family

DD Poker Community Edition supports multiplayer tournaments over LAN and Internet connections. Host your own games or join others for competitive Texas Hold'em tournaments.

Multiplayer Features

  • LAN and Internet play supported
  • Fill empty seats with AI opponents
  • Single and multi-table tournaments
  • Limit, Pot Limit, and No Limit formats
  • Real-time player chat
  • Observer mode for spectators
  • Customizable tournament structure (blinds, antes, levels, breaks, prizes)
  • Host controls: time-to-act, think bank, game options
  • 100% free - no rake, no fees

Online Lobby

  • Central chat lobby for coordinating games
  • See all connected players
  • Available before, during, and after tournaments
  • Card shortcuts in chat (e.g., /As displays Aâ™ )
  • Export and save chat logs

Self-Hosting

The Community Edition includes Docker support for hosting your own online poker server. Run private tournaments on your own infrastructure with full control over availability and configuration.

See the GitHub repository for deployment instructions.
